#                  What it is?

It is the door by which 2 PCs may communicate with each other:
```
    PC1             PC1     
+----------+    +----------+
| Socket-1*------*Socket-2 |
|          |    |          |
+----------+    +----------+
```

#                  Layers

It uses:
* Application Layer
* Transport Layer - developer might choose whether to use TCP or UDP and other minor stuff
