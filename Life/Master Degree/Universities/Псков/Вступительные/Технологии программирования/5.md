# Операторы ветвления и цикла. 

2 вида ветвления (`if`):
- if
- switch

# Метки. 

```cpp
goto labelName;

labelName:
    cout << "Hello World";
```

# Безусловный переход

- `break` - выход из switch, for, while, do
- `return` - выход из функции
- `continue` - следующая итерация цикла
- `goto метка` - безусловный переход
